# 유타 - MVP

간단한 발권 타이밍 어드바이저 MVP입니다. 두바이유(Dubai crude) 일별 가격을 내장 데이터로 계산하며, 선택한 발권일 기준으로 "지금 발권이 유리한지 / 기다리는 게 유리한지 / 중립인지"를 유류비 추세 관점에서 보여줍니다.

주요 기능
- 오피넷에서 CSV 업로드(오피넷의 Dubai 일별 가격 CSV 사용)
- 발권예정일(`이 날짜에 발권한다면?`) 선택 시 즉시 재계산
- 직전 산정기간 평균과 현재 진행중인 산정기간 평균 비교
- 변화율, 신뢰도, 권고 문구 제공

데이터 출처 및 주의
- 본 서비스는 오피넷/한국석유공사에서 제공하는 국제유가 데이터 중 Dubai 가격을 활용합니다.
- Dubai 가격은 실제 항공사 유류할증료 산정 원본이 아닌, 유류비 방향성을 보기 위한 공개 proxy 지표입니다. 결과는 참고용이며 실제 결제 전 항공사/예매처의 금액을 확인하세요.

로컬 실행
1. 의존성 설치
```
npm install
```
2. 개발 서버 실행
```
npm run dev
```

오피넷 CSV 업로드 방법(간단)
1. 오피넷 접속
2. 유가관련정보 → 국제유가 → 원유
3. Dubai 선택, 기간을 일간으로 선택
4. CSV 저장 후 앱에서 업로드

계산 방식 요약
- 현재 발권월 기준 평균: 발권월의 전전월 16일 ~ 전월 15일 평균
- 다음 발권월 예측(부분): 전월 16일 ~ 발권예정일까지의 평균(단, 실제 데이터가 없으면 최신 확보일까지만 사용)
- 변화율 = (nextPartialAverage - currentAverage) / currentAverage * 100
- 판단 기준: >=+7% => `BUY_NOW`, <=-7% => `WAIT`, 그 사이 => `NEUTRAL`
- 신뢰도: 다음 발권월 전체 산정기간 대비 확보된 데이터 진행률(30% 미만 낮음, 30-70 보통, 70 이상 높음)

향후 개선
- 오피넷 자동 다운로드/스케줄링
- 항공사별 유류할증료 고시표 추가
- 거리구간별 실제 금액 표시
- USD/KRW 환율 반영
- 원화 환산 Dubai 추세 계산
- 목적지 데이터 전체 확대

이번 버전 설명
- 유타는 두바이유 추세 기반 발권 타이밍 참고 앱입니다.
- 사용자는 국가/도시/공항과 발권일을 선택합니다.
- 발권일은 “이 날짜에 발권한다면?” 기준으로 계산됩니다.
- 노선별 운항거리 자료를 기반으로 선택 목적지의 거리와 거리구간을 표시합니다.
- 목적지 선택은 거리구간 표시용이며, 추천 알고리즘은 두바이유 추세 기준입니다.
- 실제 유류할증료 금액 계산은 아직 하지 않습니다.
- 원화 환산과 환율 반영은 이번 작업 범위가 아닙니다.
- 목적지 선택 및 거리구간 표시 추가

배포 (Vercel)
1. GitHub에 푸시되어 있는 리포지토리(이미 푸시됨)를 Vercel에 연결합니다.
2. Vercel → New Project → `day27N/0708` 선택 → Project Name 입력 → Next.js로 자동 감지됩니다.
3. Environment Variables(환경변수)를 추가합니다(대시보드 → Settings → Environment Variables):
	- `NEXT_PUBLIC_SUPABASE_URL`
	- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
	- `NEXT_PUBLIC_GA_ID`
4. Deploy 버튼을 눌러 배포합니다. 배포가 완료되면 제공된 도메인에서 서비스 확인 가능합니다.

환경변수 예시 파일: `.env.example`(리포지토리 루트)에 템플릿이 포함되어 있습니다.

Supabase 연동 및 Google Analytics 연결은 배포 후 Vercel 환경변수로 값을 추가하면 됩니다.
